# Wayflow Samples
Collection of examples on Wayflow agentic framework

## A Quoting Sales Agent with WayFlow and Oracle AI DB Similarity Search

The [quoting_sales_agent.py]("quoting_sales_agent.py") script integrates Oracle DB 26ai as vector store to support similarity search for products details finding.

## A NL2SQL Agent for Oracle DB in WayFlow with the SQLcl MCP server

The [howto_mcp_sqlcl.py]("howto_mcp_sqlcl.py") script integrates Oracle DB 26ai via the **SQLcl** MCP tool to run text-to-sql resquest by the agent.
